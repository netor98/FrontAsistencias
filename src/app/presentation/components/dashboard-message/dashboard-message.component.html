Bienvenido al Sistema de Asistencia

Controla y registra la asistencia de profesores a sus clases de forma sencilla y segura. Utiliza las distintas funcionalidades del sistema para ver tus registros, gestionar usuarios y reportes.

Profesores

Accede y visualiza el registro de asistencia de cada profesor.

Grupos

Consulta y administra los grupos y horarios asignados.

Reportes

Genera reportes comparativos de asistencias y revisa discrepancias.
